# BMI Calculator (Node.js + Express)

## How to run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open in browser:
   - http://localhost:3000

## Routes
- `GET /` — HTML form
- `POST /calculate-bmi` — calculates BMI and shows a styled result page

## Notes
- Static files (CSS) are served from `/public`.
- HTML page is in `/views`.
